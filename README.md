# 🤖 AI Medical Advisor

An AI-powered healthcare support system that provides preliminary symptom guidance, doctor discovery, and healthcare assistance before professional medical consultation.

## 📌 Overview

AI Medical Advisor is a healthcare-support project that combines Artificial Intelligence (AI) and Machine Learning (ML) to assist users in understanding potential health concerns based on reported symptoms. The system is designed to provide preliminary guidance while emphasizing consultation with qualified healthcare professionals.

In addition to AI-assisted symptom analysis, the platform supports doctor discovery, appointment booking, pharmacy workflows, and administrative management, creating a centralized healthcare support solution.

> **Disclaimer:** This project is intended for educational purposes and does not replace professional medical advice or diagnosis.

---

## 🚀 Features

- 🤖 AI-assisted symptom analysis
- 🩺 Preliminary health guidance
- 👨‍⚕️ Doctor search and appointment support
- 💊 Pharmacy support workflow
- 📋 Healthcare record management
- 🔐 Admin dashboard
- 📱 User-friendly interface

---

## 🛠️ Technologies Used

- Python
- Machine Learning
- Artificial Intelligence

---

## 👥 Stakeholders

- Admin
- User
- Doctor
- Pharmacist

---

## 🎯 Project Objective

The primary objective of AI Medical Advisor is to improve access to early healthcare guidance by integrating AI-assisted symptom evaluation, doctor booking, and healthcare management into a single platform.

# 🏛️ MCCMS - Municipal Complaint Management System

A web-based complaint management platform that enables citizens to report municipal issues while helping authorities efficiently manage and resolve complaints.

## 📌 Overview

MCCMS is designed to streamline communication between citizens and municipal authorities by providing a centralized complaint management system. Citizens can submit complaints, monitor their progress, and receive updates, while administrators can efficiently review, assign, and resolve reported issues.

The platform improves transparency, accountability, and service delivery within municipal departments.
