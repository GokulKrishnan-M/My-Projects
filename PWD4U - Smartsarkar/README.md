# 🏗️ PWD4U - Smartsarkar

A Python-based Public Works Department management system designed to simplify project administration, complaint handling, and departmental record management.

## 📌 Overview

PWD4U is a departmental management solution that assists Public Works Departments in organizing project information, handling complaints, managing contractor records, and maintaining centralized operational data.

The system reduces manual work while improving accessibility and efficiency across departmental workflows.

---

## 🚀 Features

- 🏢 Public Works project management
- 👷 Contractor management
- 📋 Complaint handling
- ⭐ Service rating system
- 📍 Location-based request management
- 📂 Centralized record storage

---

## 🛠️ Technologies Used

- Python
- HTML
- JavaScript
- CSS
- MySQL

---

## 👥 Stakeholders

- Admin (PWD Department)
- Staff
- User

## 🎯 Project Objective

PWD4U focuses on reducing manual effort within public works departments by providing a centralized platform for project, complaint, and departmental information management.
